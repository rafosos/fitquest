História Do Projeto

Este projeto tem como objetivo desenvolver um aplicativo gamificado para incentivar a prática de exercícios físicos, transformando os treinos de academia em desafios diários e competições entre amigos. Ele busca solucionar um problema comum: a falta de motivação para manter uma rotina de exercícios, frequentemente causada pela monotonia das atividades. A proposta utiliza gamificação como estratégia, permitindo que os usuários avancem com personagens virtuais à medida que completam treinos reais, tornando a experiência mais divertida e recompensadora. Este projeto se destaca por promover hábitos saudáveis de forma inovadora, incentivando um estilo de vida ativo de maneira envolvente e motivadora.
Este projeto é relevante tanto no âmbito acadêmico quanto na prática, pois une tecnologia e saúde de maneira inovadora. No campo acadêmico, ele contribui para o estudo da gamificação aplicada ao bem-estar, investigando como elementos de jogos podem melhorar a adesão a exercícios físicos. Essa abordagem oferece avanços significativos para áreas como psicologia, educação física e ciência da computação. Na prática, o projeto apresenta uma solução eficiente para motivar as pessoas a manterem uma rotina de treinos, tornando as atividades físicas mais atraentes e dinâmicas.
